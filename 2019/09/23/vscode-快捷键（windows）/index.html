vscode 快捷键（windows）
Post：2019-09-23 22:38:19 Tags：/ 工具 / Visit：

【Ctrl + Shift + P】 打开命令面板
  【Ctrl + U】 返回上一个光标位置
  【Home/End】 光标移动至行首或行尾
  【Ctrl + Home/End】 光标移动至文档的开头或末尾
  【Ctrl + D】 选中单词（第二次按下，会找到文档中下一次出现该单词的位置 并选中单词）
  【Alt + 上下】 代码移动
  【Ctrl + /】 行注释
  【Alt + Shift + F】 格式化整个文档
  【Alt + 鼠标点击】 多光标
  【Alt + Shift + I】 在选中的多行代码的 每一行 （已选中内容）后边都会添加光标
  【Ctrl + P】 快速打开文件
  【Ctrl + G】 快速跳转（代码行）
  【Ctrl + Shift + O】 符号跳转（符号：文件名、函数名、css类名）
  【F12】 跳转至 函数定义处
  【Ctrl + F12】 跳转至 函数实现处
  【Shift + F12】 引用跳转（先将光标移动到 函数/类 上）
