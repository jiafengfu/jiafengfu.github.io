类选择器

  • 通过设置元素的 class属性，为元素指定类名；
  • 文档中可为 多个元素 指定相同的类名；
  • 类选择器以 英文句号（.） 开头


id选择器

  • 通过设置元素的 id属性，为元素指定id；
  • 每个 id在文档中必须 唯一；
  • id 选择器以 井号（#） 开头。

选择器优先级

  1. 多个规则 为同一元素 指定相同属性，则拥有 更高确定度 的选择器 优先级更高；
  2. 可将多个选择器组合 来提高选择器的确定度；
  3. 若元素的CSS规则冲突，且它们的确定度相同，则 后出现的规则 优先级高。
